# pandas-assignment-solutions
Data analysis exercises in Pandas focused on real-world reporting: cleaning missing values, DataFrame transformations, and grouped salary/stakes summaries.

## Run on Google Colab

1. Open Google Colab: go to `colab.research.google.com`
2. Click **New Notebook** (or upload this project’s `.ipynb` if you have one).
3. Upload your dataset:
   - In Colab, go to the left sidebar → **Files** → **Upload**
   - Upload your `.csv` file (example: `data.csv`)
